# fosscafe-hyperledger
hyperledger fabric exploration

# software

Please install the following software on your GNU/Linux system

* docker
* docker-compose
* docker-machine (optional)

# how to setup

run the command following command
```
$ cat install-hlfv1.sh | bash
```

or execute the script 
```
do_hyperledger_fabric_install.sh
```


# start / stop the fabric

please refer to the following scripts
```
do_hyperledger_fabric_start.sh
do_hyperledger_fabric_stop.sh
```
